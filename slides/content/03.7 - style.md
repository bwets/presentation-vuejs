## Style
- Supports CSS or any stylesheet language (SASS, LESS, STYLUS) via webpack
- Scoping

``` html
<template>
    <div>
        <a href="/home">Home</a>
    </div>
<template>
<script>
    export default {
        ...
    }
</script>

<style lang="sass" scoped>
    a {
        color:red;
    }
</style>
```
