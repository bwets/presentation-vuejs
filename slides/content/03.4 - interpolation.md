## Interpolation
- Plain text
``` json
{{ variable }}
```
- attribute
``` json
<a v-bind:href="url"> ... </a>
```
- Integration of javascript
``` json
{{ ok ? 'YES' : 'NO' }}
<div v-bind:id="'list-' + id"></div>
```
