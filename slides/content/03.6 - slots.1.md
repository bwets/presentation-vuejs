# Slots
 - Provide customizable regions
 - Create container components
 - Can access children components via this.$children

``` html
<template>
    <div class="menu">
        <div class="header">
            <slot name="header" /> <!-- named slot -->
        <div>
        <div class="content">
            <slot />  <!-- default slot -->
        <div>
    <div>
</template>
```
