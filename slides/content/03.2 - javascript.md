## Javascript Components

``` html
<template>
  <div>
    <p>{{ firstname }} {{ lastname }}</p>
    <p>{{ fullname }}</p>
  </div>
</template>

<script>
  export default {
    data: function() {
      return {
        firstname: 'John',
        lastname: 'Doe'
      }
    },
    computed: {
      fullname: function() {
        return this.firstname + ' ' + this.lastname;
      }
    }
  }
</script>
```

:::
## Javascript Components

``` js
import ComponentA from './ComponentA'
import ComponentC from './ComponentC'

export default {
  components: {
    ComponentA,
    ComponentC
  },
  // ...
}
```

LC
